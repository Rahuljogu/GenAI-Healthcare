# 🏥 AI-Powered Healthcare Assistance Platform

An AI-powered healthcare web application built using **Python, Django, HTML, CSS, Bootstrap, PostgreSQL, and Google Gemini API**. The platform helps users analyze symptoms, obtain drug information, and interact with an AI medical assistant through a secure authentication system.

🔗 https://genai-healthcare-xsnp.onrender.com

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Rahuljogu/GenAI-Healthcare.git
```

Move into the project directory

```bash
cd GenAI-Healthcare
```

Create a virtual environment

```bash
python -m venv venv
```

Activate the virtual environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run database migrations

```bash
python manage.py migrate
```

Start the development server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
